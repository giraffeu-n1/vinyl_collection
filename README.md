# Коллекция винила

Django-сайт для каталогизации виниловых альбомов: обложки, группа, название, описание, поиск и авторизация.

## Возможности

- Каталог альбомов с обложками
- Поля: группа/исполнитель, название альбома, описание, изображение обложки
- Поиск по названию группы и названию альбома
- Регистрация, вход и выход
- Добавление, редактирование и удаление альбомов (только владелец)
- Фиолетовое оформление интерфейса

## Установка и запуск

```powershell
cd vinyl_collection
pip install -r requirements-dev.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Откройте в браузере: http://127.0.0.1:8000/

Админ-панель: http://127.0.0.1:8000/admin/

## Импорт фотографий с диска

```powershell
python manage.py import_vinyl "C:\music\vinyl"
python manage.py import_vinyl "C:\music\vinyl" --clear   # очистить и импортировать заново
```

Фото группируются по времени съёмки (пауза > 33 сек = новый альбом). Названия распознаются с обложки через OCR. У одного альбома может быть несколько фотографий (лицевая/оборотная сторона и т.д.).

## Структура

- `collection` — приложение с моделями, формами и страницами
- `vinylsite` — настройки проекта
- `media/` — загруженные фотографии альбомов
